freelancehunt.com
=================

Программа предназначена для оперативного получения информации
с сайта http://freelancehunt.com/

Работает под семейством ОС Windows XP / Seven / 8.

Для работы необходимо .NET FrameWork 3.5.

В файле freelancehunt.zip находится все необходимое для запуска ПО.

Если у Вас еще нет идентификатора и секретного ключа, то их можно
получить перейдя по ссылке http://freelancehunt.com/my/api

Для обсуждения проекта перейдите по ссылке:
http://freelancehunt.com/project/show/freelancehunt-prostoe-prilozhenie-pod-windows/31230.html
в разделе "Обсуждение".

Программа написана на языке C#.
